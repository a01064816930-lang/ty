마인크래프트 베드락 기반 국가전쟁 렐름 서버

💬 공식 오픈톡 참여하기

⚔️ 서버 핵심 시스템

  • 신호기 설치 시 주변 5~10칸 범위 원형 영토 자동 점령
  • 이미 점령된 영토는 다른 플레이어가 신호기를 설치해 점령 가능
  • 영토 내 신호기 파괴 시 점령권 변경 가능
  • 영토 클릭 시 소유자 닉네임 표시
  • 허락 시 타 플레이어 영토 진입 가능 (권한 시스템)
  • 신호기 클릭 후 전쟁 선포 버튼으로 국가 간 전쟁 가능

🏳️ 국가 규칙

  • 국가 생성 자유 (이름, 국기 설정 가능)
  • 동맹 및 전쟁 자유
  • 영토 확장 가능

⚠️ 서버 규칙

  • 핵 / 치트 / 불법 모드 사용 금지 (즉시 제재)
  • 무단 약탈 및 서버 악용 금지
  • 운영자 판단 절대 우선
  • 비매너, 욕설 과도 사용 금지
  • 스폰 및 보호구역 훼손 금지

📌 안내

  • 본 서버는 국가 간 전쟁과 성장 중심 서버입니다.
  • 모든 분쟁은 운영진 판단에 따라 처리됩니다.
